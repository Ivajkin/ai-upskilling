Программа: 2 недели

3 живых созвона + async-поддержка в Telegram. Не теория — практика на твоём коде, с твоими задачами.

01

Setup & First Kill

Настраиваем Cursor + Claude Code под твой проект. Конфигурим правила, MCP-серверы, шаблоны промптов. Берём реальную задачу с работы и решаем через агента — ты видишь разницу в первый час.

02

Deep Dive

Разбираем сложные кейсы: рефакторинг, архитектурные решения, тесты, code review через AI. Работаем с сопротивлением — когда AI выдаёт мусор и хочется всё бросить. Паттерны, которые работают для iOS/Swift.

03

Autonomy

Ты работаешь самостоятельно неделю. Я на связи в Telegram. Разбираем результаты: сколько % кода через агентов, где застрял, что дальше. Выпускной: у тебя workflow, который работает каждый день.
